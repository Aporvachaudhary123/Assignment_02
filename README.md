# Assignment_02
Car Dodger Game
This is a simple console game made in C language.
You control a car (A) and must avoid obstacles (X).
Move left or right using arrow keys.
If the obstacle hits the car, the game ends.

How to Play
Press Left Arrow → Move Left

Press Right Arrow → Move Right

Avoid the falling X

Score increases each time you dodge an obstacle

Press R to restart after Game Over

Press Q to quit

Requirements
Windows OS

GCC (MinGW) or any C compiler supporting <conio.h> and <windows.h>

How to Run
Save the program as CarDodger.c

Compile using:

gcc CarDodger.c -o CarDodger
Run the game:

CarDodger.exe
